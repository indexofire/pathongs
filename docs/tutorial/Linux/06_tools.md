!!! Abstract "内容介绍"
    本节主要介绍 Linux 系统下常用的管道工具。
    - `sort`: 排序
    - `uniq`: 合并重复项
    - `cut`: 按字段剪切
    - `split`: 字符串剪切
    - `tr`: 字符串替换
    - `join`: 字符串连接
    - `paste`: 将文件按列复制
    - `diff`: 比较2个文件差异
    - `parallel`: 并行处理程序
    - `curl`: 下载数据
    - `cvskit`: 命令行cvs工具
